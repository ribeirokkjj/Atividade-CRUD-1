# Atividade CRUD 1 

O objetivo dessa aplicação é criar um login usando PHP e xampp que permita também a criação de novos usuários dentro do site por meio de um cadastro de usuários

Tecnologias utilizadas: XAMPP, HTML, PHP, CSS, SQL

Criamos o index.html como login de usuários, e o home.php para o cadastro de novos usuários. Dessa forma, há também components que são formas de evitar repetições de código e há a aba de infra, onde há informações sobre o sql e informações relacionadas a conexão com o db. Em suma, todo o código foi organizados em pastas, como forma de boas práticas

O código usa o banco de dados integrado com PHP e XAMPP para poder verificar se o login existe e também para poder adicionar novos usuários com o cadastro

Aprendi sobre melhores detalhes relacionados ao PHP, que é uma ferramenta bem nova para mim. Algumas funções do PHP ainda são bem complicadas para mim, então essa prática ajudou a aprender certas partes do PHP
